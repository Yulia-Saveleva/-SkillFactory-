# -SkillFactory-

Целью проекта является демонстрация возможностей автоматизации тестирования с помощью Python, pytest и selenium. 

Проект предоставляет следующий функционал:
1) Содержит примеры автотестов на Python с использованием библиотек pytest и selenium.
2) Запуск автотестов с помощью команды pytest
3) Параметризация тестовых данных с помощью декоратора @pytest.mark.parametrize
4) Использование методов find_element, send_keys, click, get, current_url и других для работы с элементами веб-страницы
5) Использование классов WebDriverWait и expected_conditions для ожидания появления или исчезновения элементов

В ходе проверки соответствия объекта тестирования заявленным требованиям использовались техники классов эквивалентности, граничных значений, pairwise.

При тестировании применялись техники тест-дизайна:
исследовательское тестирование;
пограничные значения;
эквивалентное разделение;
предугадывание ошибок;
парамертизация тестов.
